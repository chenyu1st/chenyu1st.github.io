UItraISO（软碟通）制作

下载UItraISO

https://cn.ultraiso.net/xiazai.html

  • 安装完成后点击 试用
  • 点击文件，选择打开centos的iso文件
  • 插好U盘
  • 点击顶部菜单中的 启动 选择 写入硬盘映像
  • 硬盘驱动器选择你的U盘 ，写入方式 usb+hdd+
  • 点击写入

等待制作启动盘

U盘引导

注意：如果直接在已经存在的系统上装盘可能会导致异常，建议用一块新的盘或者直接把现有的盘格式化，利用老毛桃或者其他PE系统先格式化磁盘

  • 把U盘插在电脑上
  • 开机
  • 一般按f12或者del进入bios界面设置U盘启动
  • 进入以下画面请将光标选择install 然后按e进行编辑
  • 将“LABEL=Centos\x207\x20x\86_64 quiet”的”6_64”给删除 quiet前面加上nomodeset
  • 操作后你的install信息如下：
  • 按ctrl+x安装即可进入centos安装界面

安装centos

  • 语言选择英文
  • 时区选择中国上海
  • 软件选择最小化安装
  • 配置网络和主机名，固定一个ip并设置开机自动开启网络，根据你的实际场景来
  • 磁盘分区 选择手动分区(I will configure partitioning)

分区内容如下

目录 空间
/boot 1024M
/boot/efi 200M
swap 2048M
/ 剩余所有磁盘空间

有的时候不需要分/boot/efi

  • 点击begin installation
  • 配置root pswword

等待

装好后重启即可进入centos了

报错

U盘启动时在install centos 7时报错

dracut-pre-udev[351]:modprobe :ERROR:could not insert ‘floppy’
dracut-pre-udev[351]:modprobe :ERROR:could not insert ‘edd’:No

网上找了很久

大部分说按e，修改inst.stage2=hd:LABEL=Centos\x207\x20x\86_64 quiet 为你真实U盘的设备名（/dev/sdc4或其他)

那怎么知道设备名呢

先将inst.stage2=hd:LABEL=Centos\x207\x20x\86_64 quiet改为

initrd=initrd.img linux dd quiet

按ctrl+x

可以进入主机设备界面，找到你现在的U盘的设备名

然后再重启，继续按e

修改inst.stage2=hd:LABEL=Centos\x207\x20x\86_64 quiet

为inst.stage2=hd:/dev/sdc4 quiet

按ctrl+x

即可进入centos安装页面

但是我这昨天按照这个方法试了很多次，都不行

网上还有说是centos7.6这个版本不行，我后来又做了个centos7.8的，还是不行

最后将inst.stage2=hd:LABEL=Centos\x207\x20x\86_64 quiet改成

inst.stage2=hd:LABEL=Centos\x207\x20x\8 nomodeset quiet

按ctrl+x

才成功
